# 极简记账 — Codex 开发指引

## 项目概述
一款纯网页版个人/家庭记账工具。v1.0 为单文件 HTML（零依赖），v2.0 升级为 Express 后端 + 前端分离架构，支持多基金记账、Excel 导入、双人共享。

## 版本状态

| 版本 | 分支 | 形态 | 数据存储 |
|------|------|------|----------|
| v1.0 | master | 单文件 `index.html`，双击浏览器打开 | localStorage |
| v2.0 | （待开发） | Express 后端 + 前端分离 | JSON 文件（`data/db.json`） |

## v2.0 工作规则

1. **开发前**：阅读 `docs/` 下所有标准文件
2. **开发后**：更新 `devlog/YYYY-MM-DD.md`
3. **后端**：Express + JSON 文件存储，入口 `server.js`
4. **前端**：`public/index.html`，CSS 变量统一定义在 `:root` 中
5. **API**：所有数据操作通过 `fetch()` 调用后端 REST API
6. **零外部依赖原则**：除 express/cors 外不引入其他框架或库
7. **Git**：每阶段完成后提交

## 数据模型（v2.0）

```js
Bill {
  id, createdAt,
  fund:     "基础基金"|"育儿基金"|"教育基金"|"备用基金"|"小羡收入",  // 必填
  type:     "income"|"expense",
  categoryId: "",    // 仅支出时使用
  amount:   number,
  person:   "杨"|"欢"|"",
  date:     "YYYY-MM-DD",
  note:     ""
}
```

## 关键约定速查

- **配色**：收入绿 `#22c55e`，支出橙红 `#f97316`
- **基金配色**：基础 `#3b82f6`、育儿 `#ec4899`、教育 `#8b5cf6`、备用 `#f59e0b`、小羡 `#10b981`
- **日期格式**：`YYYY-MM-DD`
- **基金 5 个固定**，不可自定义
- **人员**：杨、欢，选填
- **记账流程**：先选基金 → 再选收入/支出 → 支出时选子分类 → 金额 → 人员 → 日期 → 备注

## Excel 导入规则

- 源文件：`D:\AI\jtjz\账本.xlsx`
- 5 基金 × 月度汇总，含杨/欢/其他补贴/工资/支出列
- 解析后生成 Bill 记录，支出统一归入"其他支出"分类
- 去重：同月+同基金+同类型+同人员 → 覆盖
