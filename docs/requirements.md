# 需求规格说明书 — 极简记账

## 1. 产品定位
一款运行在 Windows 上的个人极简财务记账软件，主打轻量化、操作直观。以网页形式提供，双击浏览器打开即可使用，无需安装。

## 2. 目标用户
- 零基础的普通用户
- 有日常个人收支记录需求的个人
- 不需要复杂财务功能，只关注流水记录与查询

## 3. 功能需求

### 3.1 记账
- **FR-01**：用户可选择记录类型为"收入"或"支出"
- **FR-02**：用户输入金额（正数，最多两位小数）
- **FR-03**：用户从预设分类中选择一个分类
- **FR-04**：用户选择日期（默认当天）
- **FR-05**：用户可选填备注文字
- **FR-06**：提交时进行表单验证（金额必填且为正数、分类必选）

### 3.2 账单列表
- **FR-07**：以卡片形式展示所有账单，按日期倒序排列
- **FR-08**：收入卡片使用绿色系标识，金额前显示 `+`
- **FR-09**：支出卡片使用橙红色系标识，金额前显示 `-`
- **FR-10**：每张卡片显示：分类图标/名称、金额、日期、备注
- **FR-11**：支持按月份筛选账单
- **FR-12**：支持按分类筛选账单
- **FR-13**：支持按备注关键词搜索账单
- **FR-14**：支持删除单条账单（需确认）
- **FR-15**：支持编辑已记录的账单

### 3.3 月度统计
- **FR-16**：顶部展示当月总收入、总支出、结余三张概览卡片
- **FR-17**：以进度条/柱状图展示各分类消费占比
- **FR-18**：支持切换查看不同月份的统计数据

### 3.4 分类管理
- **FR-19**：查看所有收支分类列表
- **FR-20**：新增自定义分类（名称 + 类型：收入/支出）
- **FR-21**：编辑已有分类名称
- **FR-22**：删除分类（若该分类下有账单则提示用户）

### 3.5 数据管理
- **FR-23**：所有数据存储在浏览器 localStorage
- **FR-24**：首次使用时自动初始化预设分类
- **FR-25**：提供清除所有数据的功能（需二次确认）

## 4. 预设分类

### 收入分类（5类）
工资、兼职、投资收益、红包、其他收入

### 支出分类（9类）
餐饮、交通、购物、住房、娱乐、医疗、教育、人情、其他支出

## 5. 非功能需求

- **NFR-01**：零外部依赖，单文件 HTML 即可运行
- **NFR-02**：完全离线可用，无需网络
- **NFR-03**：兼容 Chrome / Edge / Firefox 近两年版本
- **NFR-04**：界面响应式，桌面端优先，兼顾小屏
- **NFR-05**：操作响应时间 < 100ms（本地存储，无网络延迟）

## 6. 不包含的功能（明确排除）
- 用户账户/登录系统
- 云同步/多设备同步
- 预算设置与预警
- 数据导出 Excel/CSV
- 多账户管理（现金/银行卡等）
- 定期/周期性账单
- 图表库依赖的复杂可视化
