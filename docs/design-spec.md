# 设计规范 — 极简记账

## 1. 设计理念
- **极简**：减少视觉噪音，信息层级清晰
- **直观**：一眼能区分收入与支出，操作路径短
- **清新**：浅色背景为主，绿色表收入、橙红表支出

## 2. 配色方案

## 7. 组件设计规范

### 7.1 概览卡片（Summary Card）
- 宽高比约 3:2，最小宽度 160px
- 浅灰底 + 左侧色条（收入绿 / 支出红）
- 上方为金额数字（大号加粗），下方为标签
- 结余卡片根据正负值显示绿色或红色

### 7.2 账单卡片（Bill Card）
- 白色背景，左侧 3px 色条：绿色 = 收入，橙红 = 支出
- 卡片内布局：左侧 emoji + 分类名，右侧金额（`+`/`-` 前缀）
- 第二行：日期 + 备注
- 卡片间间距 8px
- 悬停时阴影加深
- 右上方有删除/编辑小按钮（悬停显示）

### 7.3 Tab 导航
- 水平排列，4个Tab：记账 / 账单 / 统计 / 分类
- 当前激活Tab：底部2px色条 + 文字加粗
- 非激活Tab：灰色文字，无底条

### 7.4 表单
- 输入框：白色底，边框 `--color-border`，圆角 `--radius-sm`，focus 时边框变为主色
- 下拉选择：与输入框统一风格
- 主按钮：收入场景用绿色，支出场景用橙红，通用操作用深色
- 次要按钮：浅灰底 + 边框

### 7.5 进度条（统计用）
- 高度 12px，圆角 6px
- 底色浅灰，填充色对应分类颜色（循环使用调色板）
- 右侧显示百分比和金额

### 7.6 筛选栏
- 月份选择器（`<select>`）+ 分类下拉 + 搜索框
- 水平排列，间距适中
- 搜索框带放大镜图标占位

## 8. 布局

### 整体布局
```
┌──────────────────────────────────────┐
│  顶栏：标题 + 月份选择器              │  高度 56px
├──────────────────────────────────────┤
│  概览区：3张概览卡片（水平排列）       │  高度约 100px
├──────────────────────────────────────┤
│  Tab导航：4个Tab                     │  高度 44px
├──────────────────────────────────────┤
│                                      │
│  内容区（根据 Tab 切换）              │  剩余高度
│                                      │
└──────────────────────────────────────┘
最大宽度：720px，居中显示
```

## 9. 交互细节

- 点击删除：弹出确认对话框（自定义模态框，非 `confirm()`）
- 表单提交成功：按钮短暂显示"✓ 已保存"
- 空列表状态：居中显示插图和引导文字"还没有账单，去记一笔吧～"
- 金额输入：自动聚焦，支持小数点
