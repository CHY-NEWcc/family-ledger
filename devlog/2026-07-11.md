# 2026-07-11 开发日志

## 统计页面修复与增强

### 问题修复
- **中文乱码**：统计页面的章节标题和空状态文本出现"??"乱码，原因是文件编码损坏。已修复所有乱码位置：
  - 支出分类占比 / 收入分类占比 / 基金月度概览 标题
  - 空状态图标和文本
  - 导入模态框注释（Excel 导入预览）
  - JS 注释（设置默认日期为今天）

### 新增功能
1. **支出分类饼图**（SVG 甜甜圈图）
   - 使用原生 SVG 绘制，无外部依赖
   - 中心显示支出总额，外圈按分类占比着色
   - 右侧附分类图例（名称 + 百分比）
   - 无数据时显示空状态提示

2. **消费分析**
   - 四大指标卡片：最大支出类别、前三类合计、日均支出、储蓄率
   - 智能洞察建议：
     - 单类别占比 >50% 警告
     - 入不敷出警告（🔴）
     - 储蓄率偏低提示（🟡）
     - 储蓄率优秀鼓励（🟢）
     - 日均高消费提醒

3. **年度结存 + 总结存**
   - 年份选择器（自动填充有数据的年份）
   - 年度收入、年度支出、年度结存 三卡片
   - 总结存（从第一笔到现在的累计结余）
   - 年度卡片独立于月度卡片，互不干扰

### CSS 新增
- 饼图布局（`.pie-chart-wrap`, `.pie-legend`）
- 消费分析卡片（`.analysis-cards`, `.analysis-card`, `.analysis-insight`）
- 年度总结卡片（`.annual-summary-row`, `.annual-card`）
- 年份选择器（`.year-selector-row`）
- 响应式适配
